# MAMUT-routing-TDVRPTW-Mamut2026 — retired (v1)

This satellite held the v1 Mamut2026 TDVRPTW instances. The v1 family is retired and fully superseded by the unified Mamut2026 collection, regenerated from scratch on a 5-city grid with shared sidecars and harmonized naming:

**https://github.com/ANR-MAMUT/MAMUT-routing-Mamut2026**

The v1 instances remain available in this repository's history for the record, but they are no longer mounted in [MAMUT-routing](https://github.com/ANR-MAMUT/MAMUT-routing), carry no BKS maintenance, and should not be used in new experiments. No name or content overlaps with the v2 collection (v1 names carry hash suffixes; v2 names are `mamut-<city>-n<N>-<method>`-based).
